# Checklist Técnico (Antes do Deploy)

1. **SEO**: A página possui as tags `title`, `meta description` e canonical url apropriadas?
2. **Acessibilidade**: Elementos interativos são acessíveis por teclado? Existe contraste de cor adequado?
3. **i18n**: As chaves de tradução foram aplicadas (se a funcionalidade foi mexida)?
4. **Performance**: O código está sem erros graves no console e o bundle build final hidrata corretamente?
5. **Layout Responsivo**: Teste os tamanhos mobile (`sm`), tablet (`md`) e desktop (`lg`, `xl`).
