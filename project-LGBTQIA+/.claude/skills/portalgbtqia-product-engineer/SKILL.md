# Portal LGBTQIA+ Product Engineer

Bem-vindo(a) à Skill oficial de engenharia de produto do Portal LGBTQIA+ Nordeste. Esta skill deve ser usada sempre que houver necessidade de análise, desenvolvimento, refatoração, criação de novos recursos, ajustes visuais ou adequações técnicas no portal.

## Estrutura do Portal
O portal é construído em:
- React + Vite + TypeScript
- Tailwind CSS
- Firebase / Firestore

## Diretrizes de Uso
Consulte os documentos de referência contidos na pasta `references/` e aplique-os durante o desenvolvimento.

- **editorial-guidelines.md**: Princípios de escrita, inclusão e validação.
- **image-policy.md**: Regras para imagens e acessibilidade visual.
- **technical-checklist.md**: Checklist de SEO, performance, i18n e acessibilidade.

Ao criar novos conteúdos ou executar auditorias, utilize os modelos da pasta `templates/`.
