# Política de Imagens e Assets

- **Acessibilidade**: Uso obrigatório da tag `alt` em imagens significativas. Use `aria-hidden="true"` em ícones ou imagens puramente decorativas.
- **Carregamento Otimizado**: Recomenda-se o uso de formatos otimizados (WebP) e adição do atributo `loading="lazy"` para imagens que ficam abaixo da dobra (fora do viewport inicial).
- **Proporção**: Mantenha as proporções de design. Use classes utilitárias como `object-cover` quando a imagem puder ser cortada para se adequar ao componente.
