# Portal LGBTQIA+ Permanent Rules

- O projeto usa React, Vite, Tailwind CSS e Firebase.
- Todas as alterações visuais devem respeitar a paleta de cores primária e secundária, focando em diversidade, legibilidade e alto contraste.
- O roteamento é baseado no `react-router-dom` e definido em `src/router/config.tsx`.
- Regras de conteúdo: O projeto preza pela inclusão, usando linguagem inclusiva e componentes acessíveis (a11y).
- Ferramenta de build: `npm run build` deve ser usado antes de validar mudanças estruturais.
- **Skill Associada**: Utilize a skill `portalgbtqia-product-engineer` para acessar guidelines editoriais, regras de imagem e templates.
